# Guia de Estudos: IA para Iniciantes 🤖

## 📝 Contexto e Objetivos
Este projeto foi desenvolvido como parte de um desafio da DIO para explorar o uso da IA como ferramenta de aprendizagem ativa. 
O objetivo principal é consolidar conceitos fundamentais de Inteligência Artificial para quem está começando, utilizando o NotebookLM para curadoria e síntese de conhecimentos.

### Objetivos de Estudo:
1. Compreender o que é IA e suas principais ramificações.
2. Identificar aplicações práticas de IA no dia a dia.
3. Praticar a engenharia de prompts para extrair insights de fontes técnicas.
